# Great Outdoors Style Guide

## Design Philosophy
Nature-inspired, mobile-friendly hiking experience with accessible interactions.

## Color Palette
- Forest greens for primary actions
- Earth browns for supporting tones
- Water blues for accents

## Typography
- Playfair Display for headings
- Source Sans 3 for body text
- Minimum 16px body text

## Buttons
Use clear labels:
- Select Trail
- Compare Trails
- Save Favorite

Avoid vague labels:
- Click Here
- Go
- Submit

## Spacing
4px spacing scale used throughout app.

## UI Patterns
- Rounded cards
- Soft shadows
- Mobile-first design
