# Great Outdoors вЂ” Pacific Northwest Hiking Web App

## Project Purpose

The Great Outdoors is a frontend web application designed to help hikers discover and explore Pacific Northwest hiking trails through a modern, mobile-friendly experience.

The app was created for client Evan Holt, founder of The Great Outdoors in Bend, Oregon. The goal is to provide centralized trail information, improve trail accessibility for beginners, promote hiking safety, and encourage exploration of lesser-known trails to reduce overcrowding on popular hikes.

This project is being developed as a frontend-only MVP using HTML, CSS, and JavaScript.

---

## Features

### Core Features

* Real-time trail search
* Search filtering by difficulty
* Featured Trails section
* Trail detail pages
* Mobile responsive design
* Persistent navigation bar
* Safety Tips page
* JSON-based trail dataset
* Nature-inspired UI design system

### Trail Information Includes

* Trail name
* Location
* GPS coordinates
* Distance
* Elevation gain
* Estimated hiking time
* Trail type
* Difficulty level
* Scenery photo
